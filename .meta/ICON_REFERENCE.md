## üìö Usage Guidelines

1. **Consistency**: Use the same icon for the same action across the app
2. **Color Coding**: 
   - Blue (#3b82f6) for notebooks/code
   - Purple (#a855f7) for boards/creative
   - Green (#22c55e) for notes/text
   - Red for delete/danger actions
   - Gray for secondary actions
3. **Size**: Use `w-4 h-4` for inline icons, `w-5 h-5` for buttons
4. **Import**: All icons come from `lucide-react`

```tsx
import { IconName } from 'lucide-react';

// Usage
<IconName className="w-4 h-4" />
```
